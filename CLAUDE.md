# Claude Code - Personal Website Project

## Basic Rules
- Do what has been asked; nothing more, nothing less
- NEVER create files unless absolutely necessary
- ALWAYS prefer editing an existing file to creating a new one
- NEVER create documentation (*.md) files unless explicitly requested
- ALWAYS read a file before editing it

## Stack
- Next.js 14 + TypeScript + Tailwind CSS v3 + Framer Motion + anime.js v4
- `package.json` has `"type": "commonjs"` — tailwind config must use `module.exports`
- Config files: `tailwind.config.js`, `next.config.mjs`, `postcss.config.mjs`

## Design System
- Background: `#FAF8F4` | Accent: `#ADB35B` (olive) | Text: `#1A1A1A` | Muted: `#6B6B6B`
- Warm section bg: `#F5F0E8` | Font: DM Sans

## Dev Server
- Run: `npm run dev` on port 3000
